모델 컨셉

Project SOS Purchase & Pricing Engine은 전국 어느 매물이든 면적·매매가·금융비용·관리비를 입력하면 매입 조건, 공간구성, 운영인력, 공실률 반영 가격, BEP, 공유오피스 대비 경쟁력을 자동 산출하는 표준 사업성 모델입니다.

원가절감 설계

Input

계약면적 또는 전용면적 중 하나만 입력해도 됩니다. 계약면적 입력 시 전용률 기준으로 전용면적과 공간구성이 자동 계산됩니다.

비용 구조

이 탭은 운영 원가와 초기 구축비만 봅니다. 매매가, 대출, 이자, 보유비는 Purchase Test와 Site Profitability에서 별도로 판단합니다.

가격표

최소 대여는 반일입니다. 가격은 (월 고정비 + 기업이윤)을 보수 목표 이용시간으로 나눈 뒤 룸 면적과 화상회의장비 유무로 배분합니다.

1개월 기준 비용 비교

Project SOS 1개월 이용료와 공유오피스 월 좌석 비용을 같은 인원 기준으로 비교합니다.

여의도 스파크플러스·판교 패스트파이브 가격 비교

사용자가 확인한 가격을 Input의 고급 설정에 입력하면, SOS 주력 상품과 경쟁사 월 좌석가·회의실 가격을 같은 기준으로 비교합니다.

여의도·판교 수익성 검증

인테리어비 10년 회수와 저비용 운영 구조를 적용해 여의도와 판교 후보지를 같은 기준으로 비교합니다.

공실률별 가격 시나리오

공실률이 높을수록 같은 월 고정비와 기업이윤을 적은 판매시간으로 회수해야 하므로 권장 가격이 올라갑니다.
