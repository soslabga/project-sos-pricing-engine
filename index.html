모델 컨셉

Project SOS는 실제 면적, 자본비용, 보유세, 관리비, 운영비, 공실률, 변동비, 목표 영업이익률을 기준으로 프로젝트룸 단가와 운영 손익을 관리하는 실행 모델입니다.

원가절감 설계

Input

계약면적 또는 전용면적 중 하나만 입력해도 됩니다. 계약면적 입력 시 전용률 기준으로 전용면적과 공간구성이 자동 계산됩니다.

비용 구조

이 탭은 운영 원가, 보유세, 자본비용, 초기 구축비를 함께 봅니다. 외부 임대료는 비교값으로만 사용합니다.

가격표

최소 대여는 반일입니다. 가격은 고정비, 변동비, 공실률, 목표 영업이익률을 반영한 필요 월매출을 기준으로 산정한 뒤 룸 면적과 화상회의장비 유무로 배분합니다.

1개월 기준 비용 비교

Project SOS 1개월 이용료와 공유오피스 월 좌석 비용을 같은 인원 기준으로 비교합니다.

여의도 스파크플러스·판교 패스트파이브 가격 비교

사용자가 확인한 가격을 Input의 고급 설정에 입력하면, SOS 주력 상품과 경쟁사 월 좌석가·회의실 가격을 같은 기준으로 비교합니다.

여의도·판교 실행 기준

여의도와 판교를 같은 비용·가격 기준으로 비교해 실행 가격과 운영 손익을 관리합니다.

공실률별 가격 시나리오

공실률이 높을수록 같은 월 고정비와 기업이윤을 적은 판매시간으로 회수해야 하므로 권장 가격이 올라갑니다.
